1) Install Tomcat server v7.0 or higher. 
2) Copy the Dynamo_DB_Web project into the work space of eclipse or any other IDE that supports java.
3) Right click on the project and click run as -> Run on server and choose Tomcat Server.
4) Tomcat server will be started and the user login screen will be displayed.
5) The User can register on the website and can buy items.
6) System requirements - Windows/Linux/Mac OS , Basic config.
